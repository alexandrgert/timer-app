## Что нового

### WebDAV — синхронизация между компьютерами

- **Отдельные кнопки «Скачать и объединить» / «Загрузить сейчас»** — на панели под фильтрами; вместо одной неоднозначной кнопки облака.
- **Периодическая проверка сервера** — настраиваемый интервал; при изменениях с другого компьютера запрос «Скачать и объединить?» / «Позже».
- **«Позже» с таймером** — повтор напоминания через 5 / 10 / 15 / 30 / 60 мин для той же версии на сервере; при новом hash на сервере — сразу новый запрос.
- **Pull-before-push** — перед загрузкой merge с сервером; meta-файл `.sync-meta.json` с SHA-256.
- **Проверка по hash файла** — periodic check скачивает `data.json` и сверяет hash (устаревший meta не скрывает реальные изменения).
- **Merge статуса задачи** — при слиянии учитываются `status` и `completed_at`: завершённая на одном компьютере не теряется; активная сессия → RUNNING.
- **Объединение сессий** — union по `session.id` с сохранением более «богатых» интервалов; корректный разбор ISO-меток с offset (`+03:00`).

> Пароль WebDAV и вебхук Битрикс24 **не попадают** в облако — только `data.json` с задачами и настройками UI.

### Список задач — десктоп

- **Даты «Создана» / «Завершена»** — read-only в шапке **развёрнутой** карточки (клик по строке); формат `ДД.ММ.ГГГГ ЧЧ:ММ` в **локальном** часовом поясе компьютера.
- **Адаптивная шапка** — при нехватке места мета переносится над счётчиками «сег. · всего».
- **Исправлено наложение UI** — при раскрытой карточке на «Все» и переключении на «Сегодня» / «В работе» строки больше не перекрываются.
- **Стабильная пересборка списка** — без `processEvents()` при смене вкладки/даты.

### Настройки

- **WebDAV** — расширенная вкладка (интервал, напоминание «Позже», sync on startup/shutdown); увеличена минимальная высота окна настроек.

## Исправления и внутреннее

- **`sync_webdav_now`** — после pull загрузка без повторного download (`push_merged_state`).
- Удалён неиспользуемый API редактирования дат задачи (`parse_task_datetime`, параметры `created_at`/`completed_at` в `update_task`).
- **CI:** GitHub Actions на Node.js 24; сборка deb / exe / macOS.
- **`.gitignore`:** `data.json`.
- Расширены тесты: merge status, WebDAV periodic/stale meta, UI карточки.

## Как обновиться

Скачайте нужный файл на [странице релиза v0.5.31](https://github.com/alexandrgert/timer-app/releases/tag/v0.5.31).

### Linux

```bash
wget https://github.com/alexandrgert/timer-app/releases/download/v0.5.31/tasktimer-link-b24-0.5.31-amd64.deb
sudo dpkg -i tasktimer-link-b24-0.5.31-amd64.deb
sudo apt-get install -f
tasktimer-link-b24
```

### Windows

Запустите `tasktimer-link-b24-0.5.31-win64.exe`.

### macOS

Распакуйте `tasktimer-link-b24-0.5.31-macos-arm64.zip` и перетащите `.app` в «Программы».  
*(Сборка CI — Apple Silicon; для Intel Mac нужна отдельная сборка `macos-x86_64`.)*
