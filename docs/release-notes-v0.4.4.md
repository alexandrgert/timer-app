# TaskTimer link B24 — версия 0.4.4

## Что нового

### Битрикс24

- **Журнал работ проекта** — поля СПА и entity type определяются с портала (`discover_worklog_config`), а не только захардкоженные `1092` / `ufCrm88*`.
- **Короткие сессии** при передаче в проект больше не округляются до `0` часов (минимум `0.01` ч).
- **Журнал transfer** (`bitrix_transfer_journal.json`) — после успешного API результат сохраняется до отметки в UI; при сбое повторная передача не создаёт дубль.

### Интерфейс и UX

- Плавающий виджет в трее снова можно **скрыть** (✕) и вернуть через **«Показать виджет»** в меню трея.
- Панель таймера для **paused**-задачи выбирает ту, что остановили **последней** (по времени окончания сессии).
- Диалог **«История»** дожидается завершения фоновой передачи в Битрикс перед закрытием.

### Производительность

- **WebDAV** при запуске больше не блокирует появление окна — синхронизация откладывается сразу после показа UI.
