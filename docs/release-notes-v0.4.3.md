# TaskTimer link B24 — версия 0.4.3

## Что нового

### Интерфейс (перенос из upstream TaskTimer)

- Новая раскладка: **боковая панель** (Задачи / Фокус / настройки) вместо вкладок.
- Страница задач с чипами **Сегодня / В работе / Все**, тёмной панелью таймера и обновлёнными стилями.
- Компактные строки задач (48px), шрифты **Inter** и **Roboto Mono**.
- Завершённые задачи в списке опускаются вниз; время в строках обновляется каждую секунду.

### Битрикс24: передача времени

- В окне **«История»** — колонка **«Передано»** и кнопка **«Передать в Битрикс»**.
- Отмеченные непереданные интервалы суммируются в **одну запись** на портал (проект → журнал работ СПА 1092, задача → учёт времени).
- Спецификация: [time-push-to-bitrix-design.md](superpowers/specs/2026-06-11-time-push-to-bitrix-design.md).

### Надёжность UI

- Исправлено падение при WebDAV-уведомлении в трее на старте.
- Панель таймера показывает **приостановленную** задачу; кнопка **«Продолжить»** возобновляет отсчёт.
- Добавлены автотесты нового UI (`tests/test_main_window_ui.py`).

### Сборка

- `build_deb.sh`: режим **OFFLINE=1** (без `pip install`) и защита от перезаписи `.deb` — версия поднимается при каждой сборке.
